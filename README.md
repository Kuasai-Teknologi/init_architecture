Don't waste your time when creating your first project to set up your own architecture. This package will help you to improve your setup architecture

## Example
```md
app/
┣ data/
┣ repository/
┣ modules/
core/
themes/
utils/
```

## Command usage
> Inside folder `lib/`
```dart bin/init_folder_architecture.dart --init=app```
